# EC2 Instance Start/Stop Automation with Discord Notifications

## Overview

This repository automates the start and stop of EC2 instances using Python, sends notifications to Discord, and runs via cron jobs or Jenkins pipelines for scheduling.

## Features

- **Automates EC2 Start/Stop**: Easily control EC2 instance state.
- **Discord Notifications**: Receive real-time notifications in Discord.
- **Cron Jobs/Jenkins**: Execute automatically based on schedules.

## Prerequisites

- **AWS Account** and **EC2 Instance**.
- **AWS CLI** installed and configured.
- **Python 3.x** and **Boto3** installed.
- **Discord Webhook URL**.
- **Jenkins** for pipeline execution (optional).
- **Cron** (for scheduling tasks on Linux servers).
